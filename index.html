Installation and Usage Instructions

Prerequisites

  • Python 3: Ensure you have Python 3 installed. Download from official Python website.
  • pip: Pip should come with Python. Check by running pip --version in your terminal.

Installation (Running from Source)

  1. Download and Extract: Download the system-health-utility.zip and extract it.
  2. Install Dependencies: Open your terminal/command prompt, navigate to the extracted system-health-utility folder, and run: 
    pip install -r requirements.txt
    (Ensure requirements.txt in the zip contains: psutil, schedule, requests)

Configuration (Dummy API)

This version uses a dummy API. You do not need to configure api_endpoint in config.ini. You can adjust reporting_interval_minutes.

Running the Utility in the Background

macOS/Linux:

nohup python main.py &

Check with ps aux | grep "python main.py" and stop with kill <PID>.

Windows:

start python main.py

Stop via Task Manager.

Accessing the Dashboard

Go to Dashboard after the utility is running.
